# MONTE CINEMA

 LIVE SITE URL: [MONTE CINEMA](https://monte-cinema-rails-production.up.railway.app/)

## Requirements
- ruby 3.1.2
- rails 7.0.3.1
- postgresql 14.5
- redis

## Setup

```
bundle install
```

### Database creation

```
rails db:create
```

### Database initialization

```
rails db:migrate
```

### Seeds

```
rails db:seed
```

## Services

Sidekiq - job queues
